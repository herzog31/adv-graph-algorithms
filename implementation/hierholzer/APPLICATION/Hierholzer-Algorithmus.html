Kantenzüge ohne doppelte Kanten

Das Eulertour Problem

Nicht immer ist die Länge eines Weges entscheidend, oft reicht es schon aus, einen passenden Weg zu kennen. Du möchtest beim Zeitungsaustragen den optimalen Weg wissen, sodass du keine Straße doppelt abgehen musst. Analog dazu nennt man das Finden eines Kantenzugs im Graphen ohne doppelte Kanten das Eulertour Problem.

Der hier vorgestellte Hierholzer Algorithmus löst das Eulertour Problem für passenden Graphen.

DO

{akt, u} ← Suche unbesuchte Kante

subtour ← subtour ∪ {u}

Wie sieht der (Pseudo-)Code des Algorithmus aus?

Eingabe: Ungerichteter Graph G=(V,E), keine oder genau zwei Knoten besitzen einen ungeraden Grad
Ausgabe: Eulertour/-weg als Liste von Knoten
BEGIN
   IF Graph ungültig THEN END
   akt = start ← beliebiger Startknoten
   subtour ← {start}, tour ← ∅
   REPEAT
      DO
         {akt, u} ← Suche unbesuchte Kante
         subtour ← subtour ∪ {u}
         akt ← u
      WHILE start ≠ akt
      tour = tour ∪merge subtour
      subtour ← ∅
      akt = start ← Knoten aus tour mit unbesuchten Kanten
   UNTIL tour ist Eulerweg/-tour
END
