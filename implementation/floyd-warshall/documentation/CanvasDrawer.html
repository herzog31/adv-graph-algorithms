Class: CanvasDrawer

CanvasDrawer

new CanvasDrawer(p_graph, p_canvas, p_tab)

Oberklasse für alle Klassen, die einen Graph auf das Canvas zeichnen.
Zeichnet Graph und Canvas regelmäßig, animiert die Legende und den Eingangsdialog.
Parameters:
Name Type Description
p_graph Graph Der Graph, der gezeichnet werden soll.
p_canvas Object jQuery Handler auf das Canvas, auf das gezeichnet wird.
p_tab Object jQuery Objekt des aktuellen Tabs.
