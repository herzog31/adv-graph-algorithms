Class: Forschungsaufgabe2

Forschungsaufgabe2

new Forschungsaufgabe2(p_graph, p_canvas, p_tab)

In Forschungsaufgabe 2 muss der Nutzer die Augmentationswege selber finden.
Da auch diese Aufgabe wieder Dinge auf das Canvas zeichnet, erweitert diese Klasse die Klasse CanvasDrawer.

Name Type Description
p_graph Graph Graph, auf dem der Algorithmus ausgeführt wird
p_canvas Object jQuery Objekt des Canvas, in dem gezeichnet wird.
